# 快速上手

本节将介绍如何在项目中使用 ndst-ui。

# 安装

# npm 安装

推荐使用 npm 的方式安装，它能更好地和 webpack 打包工具配合使用。

npm i ndst-ui -s

# CDN

目前可以通过 unpkg.com/ndst-ui 获取到最新版本的资源，在页面上引入 js 和 css 文件即可开始使用。

<!-- 引入样式 -->
<link rel="stylesheet" href="https://unpkg.com/ndst-ui/lib/ndst-ui.css">
<!-- 引入组件库 -->
<script src="https://unpkg.com/ndst-ui/lib/ndst-ui.common.js"></script>

# 引入

你可以引入整个 ndst-ui，或是根据需要仅引入部分组件。我们先介绍如何引入完整的 ndst-ui。

# 完整引入

在 main.js 中写入以下内容：

import Vue from 'vue';
import ndst-ui from 'ndst-ui';
import 'ndst-ui/lib/ndst-ui.css';
import App from './App.vue';

Vue.use(ndst-ui);

new Vue({
  el: '#app',
  render: h => h(App)
});

以上代码便完成了 ndst-ui 的引入。需要注意的是，样式文件需要单独引入。

# 按需引入

借助 babel-plugin-component，我们可以只引入需要的组件，以达到减小项目体积的目的。

首先，安装 babel-plugin-component：

    npm install babel-plugin-component -D

然后，将 .babelrc 修改为：

{
  "presets": [["es2015", { "modules": false }]],
  "plugins": [
    [
      "component",
      {
        "libraryName": "element-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
}

接下来，如果你只希望引入部分组件，比如 Button 和 Select，那么需要在 main.js 中写入以下内容：

import Vue from 'vue';
import { ndButton, ndVerifycode } from 'ndst-ui';
import App from './App.vue';

Vue.component(ndButton.name, ndButton);
Vue.component(ndVerifycode.name, ndVerifycode);
/* 或写为
* Vue.use(Button)
* Vue.use(Select)
*/

new Vue({
  el: '#app',
  render: h => h(App)
});
